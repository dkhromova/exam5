# Автоматизированные тесты

## Описание проекта
Проект содержит автоматизированные API и UI тесты. 
В написании теста использованы: Java, JUnit 4, JUnit 5, Selenide, Rest-Assured и Allure

## Структура проекта
- `pom.xml` — конфигурация Maven, зависимости и плагины для запуска тестов и генерации отчетов
- `src/test/java` — тесты (API и UI), вспомогательные классы 
- `target/` — результаты сборки, скомпилированные классы, отчеты и логи

## Импорт проекта и запуск тестов через терминал IDE

1. Склонировать репозиторий в свой проект:
  - git clone git@github.com:dkhromova/exam5.git
  - cd exam5

2. Установить зависимости:
   - mvn clean install -DskipTests

3. Запустить все тесты:
   - mvn clean test

4. Сгенерировать Allure-отчет:
   - mvn allure:report

5. Посмотреть сгенерированный отчет в браузере:
    - allure serve target/allure-results
